# sladocs

> A zero-install Markdown preview server with a [fumadocs](https://fumadocs.dev)-style UI.

Point it at a directory of `.md` / `.mdx` files and read them in your browser —
sidebar, search, dark mode, and all. No install, no config, no build step.

```bash
npx sladocs ./docs
```

See the [project README](https://github.com/ysds/sladocs#readme) for the full
feature list, configuration, and usage.

To build a static site from the same docs, use the companion CLI
[`sladocs-build`](https://www.npmjs.com/package/sladocs-build).
